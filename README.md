# 大创管理系统简介 java1249

## 项目概述

大创管理系统旨在为高校大学生创新项目提供全方位的生命周期管理。系统通过精细化的角色划分和功能设计，满足院系管理员、指导老师和学生三类用户的需求。

## 技术栈

- 后端：Spring Boot
- 前端：Vue 2 + Element UI
- 运行环境：Node.js 14
- 数据库：MySQL 5.7 或更高版本
- 开发工具：IDEA、VSCode

## 系统角色

系统定义了三类角色：院系管理员、指导老师和学生。

- 院系管理员：负责项目管理、评审组织与监督等。
- 指导老师：侧重于项目指导、评审以及项目进展的监督。
- 学生：围绕项目申报、进展报告、结项和评审参与。
